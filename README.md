
  # Professional Construction Company Website

  This is a code bundle for Professional Construction Company Website. The original project is available at https://www.figma.com/design/BemG8e4R7rIyt2g04w12B2/Professional-Construction-Company-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploy to Vercel

  This project includes `vercel.json` with Vite output settings and SPA route rewrites.

  - Import the GitHub repo into Vercel.
  - Framework preset: `Vite`.
  - Build command: `npm run build`.
  - Output directory: `dist`.
  - Deploy.
